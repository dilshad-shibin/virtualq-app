User Manual- Admin App

  1. Home
  2. User Manual

1.Download virtualQ Admin App from playstore

2.Shop registration will be doing with the details of the shop

3.Login with your mobile number

4.Enter the otp and get in to the app

5.Booking details can be viewed in the “request” button on the bottom portion

6. Select the order for processing

7.Press the confirm button for confirming the order after verification

8.Select confirmed order from confirmed button and click on Ready if the order is ready for delivery

9.Customer will get notification once order is in ready status

10.Once customer picked the order, select order from ready tab and convert the order to completed status by clicking on complete button

11. Cancelled orders can be viewed under rejected button on bottom portion.
